# 概率对战 Demo

一个基于概率分布采样的双人对战小游戏（玩家 vs 电脑），已实现为 `tkinter` 可视化程序。

## 核心规则

- 双方各自选择一个概率分布模型与参数。
- 每局各采样一次，并计算判定值：
  - `判定值 = 采样值 - 理论期望`
- 判定值更大者获胜。

这意味着双方判定值都满足 `E[判定值] = 0`，胜负只由分布形状与波动结构决定。

## 已实现分布（玩家可选）

- 均匀分布 `Uniform(a, -a)`（只选 `a<=0`）
- 正态分布 `Normal(0, sigma)`（只选 `sigma>0`）
- 指数分布 `Exp(lambda)`（自动中心化）
- 泊松分布 `Poisson(lam)`（自动中心化）
- 二项分布 `Binomial(n, p)`（自动中心化）
- 伽马分布 `Gamma(k, theta)`（自动中心化）
- 贝塔分布 `Beta(alpha, beta)`（自动中心化）

## 电脑对手

- 均值工程师（偏稳健）
- 爆发投机者（高波动）
- 离散操盘手（离散流派）
- 泊松计数师（计数流派）
- 全随机模拟器（随机分布 + 随机参数）

## 可视化流程（树形回退）

操作顺序为：

1. 锁定敌人
2. 锁定模型
3. 锁定参数
4. 开始判定（支持重复判定）

每局后可以：

- 再次判定（保持敌人、模型、参数不变）
- 返回参数选择（保持敌人、模型不变）
- 返回模型选择（保持敌人不变）
- 返回敌人选择（重选敌人，模型和参数重置）

## 运行方式

### 依赖

- Python 3.10+（推荐）
- 标准库 `tkinter`（Windows 通常默认可用）

### 启动

```bash
python probability_battle_demo.py
```
